# mkrank-hub

一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定网站或域名，仅作为通用 HTML 资源存储与展示使用。

## 项目简介

mkrank-hub 旨在集中存放各类独立制作的 HTML 页面，便于版本管理、预览和分享。页面内容涵盖工具、信息展示、静态模板等常见用途，不包含动态后端逻辑。

## 目录说明

```
mkrank-hub/
├── pages/            # 存放所有 HTML 页面文件
├── assets/           # 公共资源文件（CSS、JS、图片等）
└── README.md         # 本文件
```

- `pages/`：每个 HTML 页面为一个独立文件，文件名体现页面主题。
- `assets/`：存放页面共用的样式、脚本及媒体资源，便于维护。

## 页面归档说明

- 所有页面均为纯静态 HTML，可直接通过浏览器打开。
- 页面之间相互独立，不存在跨页面依赖。
- 提交新页面时，请将 HTML 文件放入 `pages/` 目录，并在 commit 信息中简要说明页面用途。
- 如需引用公共资源，请将文件放置于 `assets/` 对应子目录，并在 HTML 中使用相对路径引用。

## 维护说明

- 欢迎提交 Pull Request 增加新页面，或对现有页面进行改进。
- 请确保页面内容合规、不侵犯他人权益，且不包含任何形式的推广或诱导性信息。
- 仓库维护者会定期检查页面可用性，并清理长期无人维护的废弃页面。
- 如发现页面无法正常显示或存在明显问题，请提交 Issue 说明。
